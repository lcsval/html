# Trabalhando com fontes

Tipografia transmite mensagem

    - negrito
    - tamanho
    - estilo

-----------------------------------------

## Basic Font Properties

* font-family
* font-weight
* font-style
* font-size

-----------------------------------------

## Font Family

* Tipo de fonte de um elemento
* Lista de fontes e ordem de prioridade
* inclui *fallback* font

```css
p {
    font-family: "Times new Roman", Times, serif;
}
```
    font-weight: bold;
-----------------------------------------

## Font Weight

* Peso da fonte
```css
p {
    font-weight: bold;
}
```
-----------------------------------------
## Font style

* O Estilo da fonte
```css
p {
    font-style: italic;
}
```
-----------------------------------------
## Font Size

* O tamanho da fonte
```css
p {
    font-size: 18px;
}
```
